Maps

Maps are a useful data structure in JavaScript that allow you to store key-value pairs and remember the original insertion order of the keys.

1. Map Declaration

A Map can be declared using the `new Map()` constructor.


const map = new Map();

// Adding key-value pairs
map.set('name', 'Alice');
map.set('age', 25);

// Accessing values
console.log(map.get('name')); // Output: Alice

2. Map Methods

Maps come with various built-in methods to manipulate and access their elements.


const map = new Map();
map.set('name', 'Alice');
map.set('age', 25);

// Checking if a key exists
console.log(map.has('name')); // Output: true

// Removing a key-value pair
map.delete('age');
console.log(map.has('age')); // Output: false

// Getting the size of the map
console.log(map.size); // Output: 1

3. Map Iteration

You can iterate over map elements using loops such as `for...of` and `forEach`.


const map = new Map();
map.set('name', 'Alice');
map.set('age', 25);

// Using for...of loop
for (let [key, value] of map) {
    console.log(key + ': ' + value);
}

// Using forEach
map.forEach((value, key) => {
    console.log(key + ': ' + value);
});

4. Map Keys, Values, and Entries

Maps provide methods to retrieve keys, values, and entries.


const map = new Map();
map.set('name', 'Alice');
map.set('age', 25);

// Getting keys
for (let key of map.keys()) {
    console.log(key);
}

// Getting values
for (let value of map.values()) {
    console.log(value);
}

// Getting entries
for (let [key, value] of map.entries()) {
    console.log(key + ': ' + value);
}

5. Map vs Object

Maps and objects are both used to store key-value pairs, but they have some differences.


// Creating a Map
const map = new Map();
map.set('name', 'Alice');
map.set('age', 25);

// Creating an Object
const obj = {
    name: 'Alice',
    age: 25
};

// Map allows keys of any type
map.set(1, 'one');
console.log(map.get(1)); // Output: one

// Object keys are always strings
obj[1] = 'one';
console.log(obj[1]); // Output: one (as a string)

These are some basic concepts of working with maps in JavaScript. Practice is key to becoming proficient with them!
