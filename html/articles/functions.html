Functions

Creating functions is an essential part of programming in JavaScript. Functions allow you to encapsulate code into reusable blocks, making your code cleaner and easier to manage.

1. Function Declaration

A function declaration defines a named function that you can call by its name later on.


function greet(name) {
    return "Hello, " + name + "!";
}

// Calling the function
console(greet("Alice")); // Output: Hello, Alice!

2. Function Expression

A function expression assigns a function to a variable. You can also define an anonymous function and assign it directly to a variable.


const greet = function(name) {
    return "Hello, " + name + "!";
};

// Calling the function
console.log(greet("Alice")); // Output: Hello, Alice!

3. Arrow Function

Arrow functions provide a shorter syntax for writing functions and are often used in modern JavaScript.


const greet = (name) => {
    return "Hello, " + name + "!";
};

// Calling the function
console.log(greet("Alice")); // Output: Hello, Alice!

4. Arrow Function with Implicit Return

If your function has only one expression, you can omit the curly braces and use an implicit return.


const greet = (name) => "Hello, " + name + "!";

// Calling the function
console.log(greet("Alice")); // Output: Hello, Alice!

Example with Multiple Parameters

You can define functions with multiple parameters as well:


function add(a, b) {
    return a + b;
}

// Calling the function
console.log(add(5, 3)); // Output: 8

Optional Parameters

JavaScript does not have built-in support for optional parameters like some other languages. However, you can handle this by providing default values.


function greet(name = "World") {
    return "Hello, " + name + "!";
}

// Calling the function with and without an argument
console.log(greet());        // Output: Hello, World!
console.log(greet("Alice"));  // Output: Hello, Alice!

Function Scope

Variables declared inside a function are local to that function. They cannot be accessed outside of it.


function greet(name) {
    let message = "Hello, " + name + "!";
    return message;
}

// Trying to access the variable outside the function will result in an error
console.log(message); // ReferenceError: message is not defined

Using Functions as Values

Functions can be used just like any other value, passed around, and assigned to variables.


const sayHello = greet;

// Calling the function using the new variable
console.log(sayHello("Bob")); // Output: Hello, Bob!

These are some basic concepts of defining and using functions in JavaScript. Practice is key to becoming proficient with them!
